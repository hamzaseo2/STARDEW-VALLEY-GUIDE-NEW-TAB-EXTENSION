Get Started

Install in 5 Steps

Works on Chrome, Edge, Brave, Opera — any Chromium browser. No coding knowledge required.

01
Download the Extension ZIP
Download the repository as a ZIP file using the green "Code" button on this GitLab page, or clone it directly to your machine.
02
Extract the ZIP File
Extract the downloaded ZIP — you will get a stardew-extension folder with all the extension files inside.
03
Open Chrome Extensions Page
Open Google Chrome and navigate to the extensions page in your address bar:
chrome://extensions/
04
Enable Developer Mode
Toggle Developer Mode ON using the switch in the top-right corner of the extensions page.
05
Load the Extension
Click "Load unpacked" and select the stardew-extension folder. Open a new tab to see it in action! ✅
